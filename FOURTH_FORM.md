──────────────────────────────────────────────
I. PURPOSE OF THE FOURTH FORM
──────────────────────────────────────────────

The Fourth Form introduces the first **directed structure**  
that does not require:

• geometry  
• measure  
• time  
• energy  
• causality  
• mathematics  
• ontology  

It is direction **without vector**,  
tension **without force**,  
flow **without time**.

This is the first moment where the universe “leans.”

──────────────────────────────────────────────
II. ABSOLUTE CONSTRAINTS
──────────────────────────────────────────────

The Fourth Form must NOT:

1. create magnitude (would imply metric)  
2. create straightness (would imply geometry)  
3. create sequence (would imply time)  
4. create dependence (would imply logic)  
5. create agency (would imply a mind)  
6. create symmetry (would collapse into Form III)  
7. create asymmetry *too strongly* (would force hierarchy)  

It must be:

**the smallest possible directional bias.**
